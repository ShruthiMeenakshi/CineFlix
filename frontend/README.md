# CineFlix — React Frontend

This folder contains a small React frontend that reproduces the original UI.

Getting started:

1. From this folder run:

```bash
npm install
npm run dev
```

Or `npm start` (alias). The dev server runs at `http://localhost:3000`.

Notes:
- The project uses CDN links for Tailwind, FontAwesome and Swiper to keep the UI identical to the original HTML.
- Vite is used for fast dev server and HMR; keep `vite.config.js` if you need custom port.
